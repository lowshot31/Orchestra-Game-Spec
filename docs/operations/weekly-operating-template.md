# 주간 운영 템플릿

## 문서 목적

이 문서는 `창업자 + AI 팀 운영 가이드`를 실제 주간 운영에 적용하기 위한 실행 템플릿이다.

이 템플릿의 목적은 세 가지다.

1. 이번 주 우선순위를 명확히 정한다
2. 제품 개발과 회사 운영을 한 흐름으로 묶는다
3. 메인 모델과 서브 에이전트의 역할을 주간 단위로 관리한다

이 문서는 매주 복제해서 사용한다.

권장 사용 위치:

- 주간 회의 시작 전 초안 작성
- 주간 회의 중 업데이트
- 주간 종료 시 회고 반영

---

## 사용 방법

1. 이 파일을 새 주차용 파일로 복사한다
2. 주간 회의 전에 창업자가 1차 초안을 채운다
3. GPT-5.5 메인 모델과 함께 우선순위를 조정한다
4. 주간 진행 중 상태를 갱신한다
5. 금요일 또는 주간 종료 시 결과와 학습 내용을 남긴다

권장 파일명 예시:

- `2026-05-04-weekly-ops.md`
- `2026-05-11-weekly-ops.md`
